#### Anonymous config

Anonymous config is associated with neither name nor model. E.g.

[source,xml]
----
<feature-pack xmlns="urn:wildfly:pm-feature-pack:1.0"
              groupId="org.jboss.pm.test" artifactId="fp1" version="1.0.0">
  <config>
    <!-- config body -->
  </config>
[source,xml]
----
